1. С подключением к API (data-api)

Данные загружаются с сервера. Если API недоступен, используются резервные данные.
<div id="adsr-root"></div>
<script src="embed.js" data-root="adsr-root" data-api="http://localhost:4050"></script>

2. Без API (fallback данные)

Используются встроенные тестовые данные.
<div id="adsr-root-fallback"></div>
<script src="embed.js" data-root="adsr-root-fallback"></script>
